## 🚀 快速开始

```bash
# 1. 克隆/下载项目
# 2. 安装依赖
npm install

# 3. 配置环境变量 (见下文)

# 4. 部署 slash commands (首次或修改命令后运行)
node deploy-commands.js

# 5. 启动 Sky
node index.js
```

### 🔑 环境变量 (`.env`)

请确保你的 `.env` 文件包含以下配置：

```env
BOT_TOKEN=你的_Discord_Bot_Token
GEMINI_API_KEY=你的_Google_Gemini_API_Key (可选，开启AI功能)
OWNER_ID=你的_Discord_用户ID (用于主人特权命令)
```

## 🤖 智能 AI 功能

Sky 接入了 **Google Gemini AI**，它不再只是死板的回复：

- **智能聊天**：使用 `/ask` 提问，或者在频道里 **@Sky** 直接开聊。
- **个性化回复**：AI 会根据当前的模式（骚骚/正常）自动调整语气。
- **上下文理解**：Sky 能读懂你的情绪，给予最贴心的（或最 drama 的）回应。

---

## 📈 强大的绘图功能 (`/graph`)

支持多种图表展示：
- **柱状/折线/饼图**：输入标签和数据即可生成精美图表。
- **函数图像**：支持绘制数学函数，如 `x^2`, `sin(x)`, `sqrt(x)` 等。
- **自定义**：支持设置标题、坐标范围和颜色。

---

## 📅 日历与提醒系统 (`/calendar`)

再也不会错过重要时刻：
- **添加事件**：`/calendar add event:开会 time:2026-05-07 15:30 remind:10m`
- **自动提醒**：Sky 会在事件开始前通过 Embed 消息提醒你。
- **持久化记录**：随时查看你的待办清单。
